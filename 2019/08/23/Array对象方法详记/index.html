    Array对象

    1. concat
      返回新数组，不改变原数组

    2. copyWithin
      改变原数组

    3. entries
      可迭代对象可以用for … of循环来遍历

    4. every
      不会修改原数组

    5. fill
      改变原数组

    6. filter
      返回新数组，不改变原数组

    7. find
      不改变原数组

    8. findIndex
      不改变原数组

    9. forEach
      不改变原数组

    10. flat
      返回一个新数组，不改变原数组

    11. flatMap
      返回一个新数组，不改变原数组

    12. from

    13. includes

    14. indexOf

    15. isArray

    16. join

    17. keys

    18. lastIndexOf

    19. map
      返回新数组，不改变原数组

    20. pop
      改变原数组

    21. push
      改变原数组

    22. reduce

    23. reduceRight

    24. reverse
      改变原数组

    25. shift
      改变原数组

    26. unshift
      改变原数组

    27. slice
      返回新数组，不改变原数组

    28. some
      不改变原数组

    29. sort
      会改变原数组

    30. splice
      会改变原数组

    31. toString
      不改变原数组

    32. values

    注意：
    以下方法会改变原数组
    copyWithin fill pop push shift unshift reverse sort splice
    以下方法不会改变原数组
    concat flat flatMap map slice toString

    功能

    • 增添
      push unshift splice
    • 删除
      pop shift splice
    • 选取
      slice
    • 排序
      reverse sort
    • 填充
      fill
    • 遍历
      every filter forEach map some
    • 查找
      find findIndex includes indexOf lastIndexOf
