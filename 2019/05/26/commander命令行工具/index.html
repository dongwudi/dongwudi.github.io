    commander命令行工具

    测试：

    api

    1. .version
      定义命令程序版本号
      .version(<string>)

    2. .option
      定义命令选项
      .option(flags, description, [fn], defaultValue)
      flags: 分为长短标识，中间用逗号、竖线或者空格分割。标志后面可跟必须参数或可选参数，前者用 <> 包含，后者用 [] 包含

    examples:

    1. .command
      添加命令名称
      .command('name', 'description', opts)

    1. .action
      定义命令的回调

    2. .description
      定义命令的描述

    3. .parse
      用于解析process.argv，设置options以及触发commands
